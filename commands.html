Main Commands

-play [song name]/[URL]

play the music of your choice whith help of this command.

-247

Make the bot joined in the voice channel for 24/7.

-connect

Connect the bot into a voice channel.

-disconnect

Disconnect the bot from a voice channel

-loop song

Starts/Stops a loop of the current song.

-loop queue

Starts/Stops the loop of the cuurent queue.

-loop off

Stops every loop command in the server if enabled.

-radio

Start a radio in a voice channel.

-nowplaying

Checks the current song being played in the voice channel.

-pause

Pauses the current song being played.

-resume

Resumes the current paused song.

-queue

Checks the current queue of the playlist.

-shuffle

Shuffles the current queue.

-skip

Skips the current song to the next according to the queue.

-search [song name]

Searches the song you've asked for to be played.

-volume [1 - 100]

Changes the volume of the song being played in the voice channel.

-seek

Seeks into the queue and give information about it.

-remove [queue number]

Removes the song according to the queue numbers being added.

-stop

Stops the song being played and clears the queue.

-setup-dj [@any_role]

Sets a DJ role for the server DJ's.

Filter Commands

-8d

Changes the filter to 8D song.

-bassboost

Boosts the bass of the song.

-chipmuk

Changes the song to chipmuk frequency.

-karaoke

Changes the filter into karaoke system.

-rotation

Rotates the song frequency and makes it more fun.

-timescale

Makes the song slow hence making it lengthy.

-vibrato

Starts vibaration in song's base.

-clearfilter

Clears all the filters in the database.

Other Commands

-avatar [mention user]

Shows the profile picture of the mentioned user.

-kiss [mention user]

Your PFP kisses the mentioned user's PFP.

-meme

Sends a meme back in the chat.

-rip [mention user]

Sends a rip meme of the mentioned user.

-spank [mention user]

Sends a spank meme of the mentioned user.

-trigger [mention user]

Sends a trigger meme of the mentioned user.

-wasted [mentioned user]

Sends a wasted meme of the mentioned user.

-help

Sends the help command menu containing all the bot commands.

-invite

Sends the invite link of the bot back in the chat.

-ping

Sends back the ping and latency of the bot's hosting server.

-clusters

Sends the cluster information back in the chat.

-stats

Sends some cool bot statistics to you.

-uptime

Sends the bot uptime back.
